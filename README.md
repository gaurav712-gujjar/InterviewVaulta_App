# InterviewVault — Tech Interview Prep Tool

A beautiful Flask web app to browse curated interview questions
by skill and company (Infosys, Wipro, TCS, General).

## Setup & Run

```bash
pip install flask
python app.py
```

Then open: http://localhost:5000

## Skills Available
- Python
- SQL
- Machine Learning
- Data Science
- Data Analysis
- Data Engineering
- Behavioral / HR
- General Technical

## Features
- Search by skill (or use quick-access chips)
- Filter by company (Infosys, Wipro, TCS, General)
- Copy individual questions or copy all at once
- Dark, polished UI

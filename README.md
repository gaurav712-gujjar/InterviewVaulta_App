# 🚀 InterviewVault — Tech Interview Prep Tool

InterviewVault is a modern **Flask-based web application** designed to help developers prepare for technical interviews.  
It provides **curated interview questions categorized by skills and companies** such as **Infosys, Wipro, TCS, and General technical rounds**.

The app features a **clean dark UI, search functionality, and quick copy tools** to make interview preparation faster and more organized.

---

# ✨ Features

- Browse interview questions by **skill**
- Filter questions by **company (Infosys, Wipro, TCS, General)**
- **Quick search** to find relevant topics
- **Copy individual questions** instantly
- **Copy all questions** with one click
- **Modern dark UI**
- Simple **Flask backend**

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/interviewvault.git
cd interviewvault
```

Install dependencies

```bash
pip install flask
```

Run the application

```bash
python app.py
```

Open your browser and go to

```
http://localhost:5000
```
